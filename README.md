# quizzy-app
An Android application for playing quizzes in forms of Multiple-choice questions among different topics.
The application is available in 2 modes:
  - Offline: Single-player mode, modifiable settings for each attempt. Questions can be in plain text or contain images. Historic attempts of quizzes are recorded and can be reviewed using graphs, charts, ... Offline mode also allows users to create own topics or sets of questions and insert own questions and answers. 
  - Online mode: Multi-player mode, allow users to create room and compete with one other user in a shared set of questions. Additional features include a news feed to post status or share game results, a power store (set of powers to buy using coins gained from matches and can be used during the matches to gain specific benefit or attack the opponent). Questions are available in plain text, or can contain images, videos. Statistics of historic matches is also available
